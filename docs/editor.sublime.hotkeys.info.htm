    Sublime text'o šūtkatų suvestinė -- pagalbinė informacija

    Čia naudojame sutrumpinimus.

    Reikalingi, kad greitai paimti:

    sublime.log_commands(True)

    Klavišų žymėjimas:

    super == Win

    Shortcut'ų-hotkey'sų operatoriai

    / — jungia atskirus shortcut'us, įvykdančius to pačio tipo (panašaus pobūdžio) komandas, jos gali būti priešingos ar papildančios

    ctrl+up / ctrl+down

    Perkelia eilutes su pažymėjimu ir kursoriu aukštyn / žemyn

    || — jungia atskirus shortcut'us, įvykdančius tą pačią komandą

    ctrl+shift+up || shift+backspace

    Antrasis shortcut'as gali būti pakartotas ir kitoje vietoje.

    Windows'ų naudojamų hotkeys'ų arba neveikiančių klavišų kombinacijų remap'inimas Autohotkey'se

    esant aktyviam ST fiziškai paspaudus realų hotkey'są, ST gauna čia nurodytą

    Kaip-kur saugome informaciją apie hotkeys'ų remap'us:

    • Šūtkatų suvestinėje nurodome tik kokį fiziškai reikia spausti hotkeys'ą.
    • D:\mano uzrasai\win keyboard shortkey all.xlsx išsaugome abu: ir kokį fiziškai reikia spausti, ir kokį realiai programa (t. y. Sublime Text'as) gaus iš Autohotkey'so.

    super+f

    [Find > Incremental Find]

    nors realiai Sublime Text'e yra:

    { "keys": ["super+ctrl+alt+shift+f"], "command": "show_panel", "args": {"panel": "incremental_find", "reverse":false} }

    bet su Autohotkey'su (čia C:\progdata\AutoHotkey\~darbinis-naudojamas.ahk) turime priskyrę kad nuspaustas Win+F generuoja super+ctrl+alt+shift+f

    Šūtkatų kūrimo strategijos

    Visiems bendras pirmas aukštas

    Turime "visų pirmajam aukštui" skirtus klavišus, pvz.: f1 ir f2, turime šūtkatą { "keys": ["ctrl+]"], "command": "move_to" }, norime:

    • su šiuo susieto kito — { "keys": ["f1", "ctrl+]"], "command": "move_to_kitur" },
    • dar kito — { "keys": ["f2", "ctrl+]"], "command": "move_to_dar_kitur" }.

    Šį būdą naudojame taip:

    ctrl+k

    Naudojame default'iniams ST hotkeys'ams, kurie man nelabai reikalinki, pakeisti, pvz. default'iškai yra:

    { "keys": ["f6"], "command": "toggle_setting", "args": {"setting": "spell_check"} },
{ "keys": ["ctrl+f6"], "command": "next_misspelling" },
{ "keys": ["ctrl+shift+f6"], "command": "prev_misspelling" },

    Keičiame

    { "keys": ["ctrl+k", "f6"], "command": "toggle_setting", "args": {"setting": "spell_check"} },
{ "keys": ["ctrl+k", "ctrl+f6"], "command": "next_misspelling" },
{ "keys": ["ctrl+k", "ctrl+shift+f6"], "command": "prev_misspelling" },

    Unikalus pirmas aukštas

    Turime šūtkatą { "keys": ["ctrl+]"], "command": "move_to" } ir norime daugiau su šiuo susietų:

    • perdarome { "keys": ["ctrl+]","ctrl+]"], "command": "move_to" }
    • kitas — { "keys": ["ctrl+]","ctrl+["], "command": "move_to_kitur" },
    • dar kitas — { "keys": ["ctrl+]","]"], "command": "move_to_dar_kitur" }.

    Antram aukštui galima naudoti klavišus pagal:

    • prasmę: raidžių, skaičių klavišai, kitų simbolių,
    • arčiau prie pirmojo aukšto klavišų esantys klavišai.
