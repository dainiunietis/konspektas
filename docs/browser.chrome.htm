    Console panel'ė

    Console Overview  |  Chrome DevTools  |  Google Developers

    Console panel'ės atidarymas-uždarymas

    ctrl+shift+j — jeigu paspaudžiamas, kai jau yra atidaryti DevTools'ai — juos uždaro, nepriklausomai yra atidaryta ar aktyvuota Console panel'ė ar ne.

    Konsolės kintamieji — nuorodos į Elements panel'ėje pažymėtus DOM elementus

    • $0 — Elements panel'ėje pažymėtas DOM elementas

    • $1..N — prieš tai Elements panel'ėje pažymėti DOM elementai

     

    inspect(node) — pažymi DOM elementą Elements panelėje ir išveda DOM elemento nuorodą-mirror'ą Console'ėje

    inspect(node) veikia tik Console'ėje

    Puslapio screenshot'inimas

    Daro puslapio vaizdo screenshot'us.

    Kaip veikia:

    • Capture screenshot — daro viewport'o, t. y. matomos puslapio zonos, screenshot'ą — VEIKIA GERAI.
    • Capture full size screenshot — daro viso puslapio turinio pagal esamą viewport'o plotį screenshot'ą — VEIKIA GERAI.
    • Capture node screenshot — VEIKIA KREIVAI.
