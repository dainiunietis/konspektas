Favicon'o prijungimas per SVG Data-URI su Dark Mode Detection

Pagal OMG, SVG Favicons FTW!

We can add a <style> tag to our SVG and use the prefers-color-scheme media query to change our icon based on the user’s dark mode preferences.

Prijungimas HTML faile:

<link rel="icon" href="data:image/svg+xml;utf8,
<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'>
  <style>
    ... CSS rules'ai baltam-šviesiam režimui ...
    @media (prefers-color-scheme: dark)  {
      ... CSS rules'ai juodam-tamsiam režimui ...
    }
  </style>
  ... SVG kodas ...
</svg>">
