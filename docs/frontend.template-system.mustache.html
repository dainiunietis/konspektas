    Sintaksės suvestinė — cheat sheet

    Kintamasis (tag type — variable)

    Kintamojo sintaksė

    Paprasčiausias-bazinis tago tipas:

    The most basic tag type is the variable.

    Kaip veikia tag key'so hash'e radimas:

    A {{name}} tag in a basic template will try to find the name key in the current context. If there is no name key, the parent contexts will be checked recursively. If the top context is reached and the name key is still not found, nothing will be rendered.

    Template:

    Vardas: {{name}}
Metai: {{age}}

    Hash:

    {
  "name": "Petras",
  "age": 27
}

    Output:

    Vardas: Petras
Metai: 27

    HTML spec. simbolių išvedimas

    HTML spec. simboliai: < > " &

    Default'iškai HTML spec. simboliai užkoduojami:

    All variables are HTML escaped by default. If you want to return unescaped HTML, use the triple mustache: {{{name}}}.

    Template:

    Vardas: {{name}}
Aprašymas (escaped): {{description}}
Aprašymas (unescaped): {{{description}}}

    Hash:

    {
  "name": "Petras",
  "description": "apskritai <b>geras</b>"
}

    Output:

    Vardas: Petras
Aprašymas (escaped): apskritai &lt;b&gt;geras&lt;/b&gt;
Aprašymas (unescaped): apskritai <b>geras</b>

     

    ...

     

    You can also use & to unescape a variable: {{& name}}. This may be useful when changing delimiters.

     

     

    Kai hash'e nėra nurodyto tag key'so

     

    By default a variable "miss" returns an empty string. This can usually be configured in your Mustache library. The Ruby version of Mustache supports raising an exception in this situation, for instance.
