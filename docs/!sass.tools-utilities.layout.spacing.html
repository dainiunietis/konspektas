Funkcija užvardintiems spacing'o dydžiams išvesti

sass.szm/_tools.layout.spacing.scss: ✒ szm-spacing-size($sizeName, $unit: px)

  padding:       szm-spacing-size(tiny);
  margin-bottom: szm-spacing-size(tiny, em);
}

  margin-bottom: 0.4em;

Mixin'ai išvesti padding: ir margin: deklaracijas su užvardintais spacing'o dydžiais

Galima padaryti 2 būdais:

su SZM bibliotekos mixin'u

sass.szm/_tools.layout.spacing.scss:
@mixin szm-spacing-size-boxSides($property, $sizeName, $side: "", $important: "!")

div {
  @include szm-spacing-size-boxSides(margin, medium, right);
  @include szm-spacing-size-boxSides(padding, small, horizontal);
}
div {
  margin-right:  16px !important;
  padding-left:  10px !important;
  padding-right: 10px !important;
}
su projekto mixin'ais, kurie yra pirmosios SZM bibliotekos mixin'o shorthand'ai

sass/_tools.layout.spacing.scss:
@mixin u-margin($sizeName, $side: "", $important: "!")
@mixin u-padding($sizeName, $side: "", $important: "!")

div {
  @include u-margin(medium, right);
  @include u-padding(small, horizontal);
}
div {
  margin-right:  16px !important;
  padding-left:  10px !important;
  padding-right: 10px !important;
}
