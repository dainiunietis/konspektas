    Kur laikomi packages'ų failai (Package Locations)

    http://docs.sublimetext.info/en/latest/extensibility/packages.html#package-locations-and-abbreviations

    Packages'ų Locations'ų hierarchija, t. y. iš kokio Location'o naudojama, jei yra dublikatai:

    If two packages with the same name exist in both Installed Packages and Shipped Packages, the one in Installed Packages will be used and the one in Shipped Packages will be ignored.

    Any file in Packages/PackageName takes precedence over an identically named file in Installed Packages/PackageName.sublime-package or Shipped Packages/PackageName.sublime-package.

    Packages'ų Locations'ai ir jų Abbreviations'ai (Package Abbreviation), pradedant nuo žemiausio prioriteto:

    Shipped Packages

    D:\progs\Sublime Text 3\Packages — čia packages'ai saugomi ZIP'iniuose archyvuose package-pavadinimas.sublime-package, t. y. vienam package'ui vienas toks failas. Tai yra sisteminiai (core packages ir default packages) ST'o packages'si, todėl — This folder is not intended to be modified by the user.

    Installed Packages

    D:\progs\Sublime Text 3\Data\Installed Packages — čia taip pat *.sublime-package

    Packages, Override Packages

    D:\progs\Sublime Text 3\Data\Packages — o čia jau ne ZIP'ai, o katalogai skirti packages'ams, su failais juose

     

    Konkretus pavyzdys kaip override'inau ST core-default-shipped Package'so vieną iš konfigūracinių failų, tiksliau kontekstinį menu'są redaktoriuje:

    1. Paėmiau D:\progs\Sublime Text 3\Packages\Default.sublime-package, ir išarchyvavau į D:\progs_data\Sublime Text 3\Packages isarchyvuoti -- core-default-shipped\Default
    2. Failą D:\progs_data\Sublime Text 3\Packages isarchyvuoti -- core-default-shipped\Default\Context.sublime-menu nukopijavau į D:\progs\Sublime Text 3\Data\Packages\Default\Context.sublime-menu
    3. Dabar D:\progs\Sublime Text 3\Data\Packages\Default\Context.sublime-menu pilnai override'ina core-default-shipped Package'se D:\progs\Sublime Text 3\Packages\Default.sublime-package esantį Context.sublime-menu
