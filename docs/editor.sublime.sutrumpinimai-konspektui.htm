    Sutrumpinimai konspektui — Sublime Text

    ST package'ų žymėjimas

    package::packageName

    Konkretus pvz.:

    • package::FuzzyFileNav
    • instaliacija: Data\Installed Packages\FuzzyFileNav.sublime-package

    ST komandos

    [M][...]

    Meniusai, jie apibrėžiami failuose *.sublime-menu.

    Variantai:

    [M][Kažkas > Anas] arba trumpiau be prefikso [Kažkas > Anas]

    Globalus-pagrindinis ST Meniu.

    [MSB][Kažkas > Anas]

    Side Bar'o kontekstinis meniu.

    [MR][Kažkas > Anas]

    Kontekstinis Meniu Redaktoriuje.

    [CP][...]

    Ką matome [Tools > Command Palette...], t. y. kas yra failuose *.sublime-commands komandos objekto-įrašo elementas caption, pvz.:

    [CP][Kažkokia komanda] == [CP][Kažkokia komanda][kažkokia_komanda] == { "command": "kažkokia_komanda", "caption": " Kažkokia komanda" }:

    [C][...]

    Package'o Command'a, failuose *.sublime-commands, *.sublime-menu, *.sublime-keymap nurodoma objektų-įrašų elementu command, pvz.:

    [C][kažkokia_komanda]

    Be parametrų-argumentų, faile *.sublime-keymap >>> { "keys": ["ctrl+t"], "command": "kažkokia_komanda" }

    [C][kažkokia_komanda(tas:taip,anas:kitaip)]

    Su parametrais-argumentais, faile *.sublime-keymap >>> { "keys": ["ctrl+t"], "command": "kažkokia_komanda", "args": {"tas": "taip", "anas": "kitaip"} }

    [C]["command": "kažkokia_komanda", "args": {"tas": "taip", "anas": "kitaip"}]
