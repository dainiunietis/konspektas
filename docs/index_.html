    Inline elements and block-level elements

    HTML elements are usually "inline" or "block-level" elements. An inline element occupies only the space bounded by the tags that define it. A block-level element occupies the entire space of its parent element (container), thereby creating a "block."

     

    Wrapping name-value groups in div elements

    WHATWG HTML allows wrapping each name-value group in a <dl> element in a <div> element. This can be useful when using microdata, or when global attributes apply to a whole group, or for styling purposes.

    <dl>
  <div>
    <dt>Name</dt>
    <dd>Godzilla</dd>
  </div>
  <div>
    <dt>Born</dt>
    <dd>1952</dd>
  </div>
  <div>
    <dt>Birthplace</dt>
    <dd>Japan</dd>
  </div>

    :is() ir :where()

    :is() ir :where() daro tą patį — vidinį grupavimą selektoriuje, bet skiriasi kaip įtakoja selektoriaus specificity:

    • :is() — counts towards the specificity of the overall selector (it takes the specificity of its most specific argument)
    • :where() — has a specificity value of 0

    Pvz. — :is() padidina selektoriaus specificity, o :where() — nepadidina

    CSS savybėms, kurios turi neskaitines reikšmes, galima su transition-delay: nustatyti kitos reikšmės priskyrimą po nustatyto laiko

    .a20210706142334 {
  --transition-duration: 1s;
  opacity: 1;
  visibility: visible;
  /* no visibility transition */
  transition: opacity var(--transition-duration);
}
.a20210706142334--is-hidden {
  opacity: 0;
  visibility: hidden;
  /* set only a delay for the visibility property */
  transition:
    opacity var(--transition-duration),
    visibility Os var(--transition-duration);
}

    croll-behavior:

    Šita savybė įtakoja puslapio scrolling'o pobūdį, tik kai:

    • vartotojas aktyvuoja nuorodą <a href="#tas"> į tame pačiame puslapyje esantį <div id="tas">
    • Atidarius puslapį http://upas.lt/ten.html#tas
    • su JS per CSSOM scrolling APIs

    Kai scroll-behavior: pritaikomas <html>:

    When this property is specified on the root element, it applies to the viewport instead.

    O kai <body>:

    This property specified on the body element will not propagate to the viewport.
