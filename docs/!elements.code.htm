    Kodas, code — Design Style Guide

    Kodo formatavimas-spalvinimas su JS

    Kaip nurodome HTML'e programavimo kalbą:

    <pre><code class="language-css">
  ...
</code></pre>

    Nurodome HTML'e nedaryti formatavimo-spalvinimo su JS:

    <pre><code class="language-none">
  ...
</code></pre>

    highlightjs.org

    Kaip įjungiamas formatavimas-spalvinimas:
    /assets/js/master.requirejs.js # formatavimas-spalvinimas su highlightjs.org

    automatinis prog. kalbos nustatymas

    <pre><code> — čia daro kodo spalvinimą

    <div>
  <p>Agurkai šiemet auga negausiai &#8212; kodėl?</p>
</div>
    div {
  padding: 0 1em;
  border: solid red 3px;
}

.geras {
  text-align: left;
  color: blue;
}
    $plotis: 80%;

#navbar {
  width: $plotis;
  height: 23px;
  ul { list-style-type: none; }
  li {
    float: left;
    a { font-weight: bold; }
  }
}
    function RemoveText(el){
  var parent = el.parentNode;
  for(var i=0; i < parent.childNodes.length;i++) {
    var para = parent.childNodes[i];
    if(para.nodeName == "P" && para.getAttribute('class')=='added') {
      parent.removeChild(para);
      break;
    }
  }
}
