    background-blend-mode:

    css.blend-modes

    ~bg-gauti-skirtingo-intensyvumo-currentColor

    background-blend-mode: normal | multiply | screen | overlay | darken | lighten | color-dodge | color-burn | hard-light | soft-light | difference | exclusion | hue | saturation | color | luminosity

     

    Kiekvienam background'o sluoksniui galima priskirti savą blending mode:

    Blending modes should be defined in the same order as the background-image property.

    background-image: url(obuolys.png), url(kriaušė.png), url(slyva.png);
background-blend-mode: multiply, overlay, screen;

    Jei background-image: ir background-blend-mode: masyvų ilgiai nesutampa:

    If the blending modes' and background images' list lengths are not equal, it will be repeated and/or truncated until lengths match.

    background-blend-mode: multiply;
background-image: url(obuolys.png), url(kriaušė.png), url(slyva.png);
/*                multiply          multiply          multiply */

background-blend-mode: multiply, overlay;
background-image: url(obuolys.png), url(kriaušė.png), url(slyva.png);
/*                multiply          overlay           multiply */

    [ <bg-layer> , ]* <final-bg-layer>
<bg-layer> =       <bg-image> || <bg-position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box> || <box>
<final-bg-layer> = <bg-image> || <bg-position> [ / <bg-size> ]? || <repeat-style> || <attachment> || <box> || <box> || <'background-color'>

     

    <box> || <box>

    The <box> value may be included zero, one, or two times. If included once, it sets both background-origin and background-clip. If it is included twice, the first occurrence sets background-origin, and the second sets background-clip.

     

    <bg-position> [ / <bg-size> ]?

    The <bg-size> value may only be included immediately after <position>, separated with the '/' character, like this: center/80%

  [ <angle> | to <side-or-corner> ,]? <color-stop-list> )

where <side-or-corner> = [ left | right ] || [ top | bottom ]

    1-value syntax

    If only one value is specified, the second value is assumed to be center.

    background-position: center

    The keyword value center, which centers the image.

     

    background-position: left | right | top | bottom

    One of the keyword values top, left, bottom, right. This specifies an edge against which to place the item. The other dimension is then set to 50%, so the item is placed in the middle of the edge specified.

     
     
     
     

    background-position: <length> | <percentage>

    A <length> or <percentage>. This specifies the X coordinate relative to the left edge, with the Y coordinate set to 50%.

     
     

     

     

    2-value syntax

    background-origin:

    Leidžia kiekvienam background'o sluoksniui nustatyti savo pozicionavimo atskaitos sritį.

    The background-origin CSS property sets the background positioning area. In other words, it sets the origin position of an image set with the background-image property.
