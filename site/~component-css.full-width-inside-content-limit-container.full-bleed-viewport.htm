Full width inside-content limit container

The "Inside" Problem | CSS-Tricks

Reikalavimai:

You need a full-width container element because the design has a background-color that goes from edge-to-edge horizontally. But the content inside doesn't necessarily need to be edge-to-edge. You want to:

  1. Limit the width (for large screens)
  2. Pad the edges
  3. Center the content

Centered, max-width/padding containers inside full-width color bands.

.component {

  padding: 2rem 1rem;
}

.wrapper {
  max-width: <max-turinio-plotis>;

What if you're already inside a container you can't control and need to break out of it?

...

<main>
  <p>...</p>
  <section class="full-width">
    <div class="full-width-inside">
      <p>...</p>
    </div>
  </section>
  <p>...</p>
</main>
:root {
  --contentWidth: 75ch;
}

main, .full-width-inside {
  max-width: var(--contentWidth);
  margin: 0 auto;
}

.full-width {
  padding: 1rem 0;
  width: calc(100vw - 17px);
  margin-left: 50%;
  transform: translateX(-50%);
}

.full-width-inside {
  box-sizing:    content-box;
  padding-left:  .8rem;
  padding-right: .8rem;
}
