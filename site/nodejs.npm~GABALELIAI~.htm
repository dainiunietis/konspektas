    Kaip atnaujinti patį npm'ą

    http://stackoverflow.com/questions/6237295/how-can-i-update-node-js-and-npm-to-the-next-versions

    Skirtingi būdai:

    1. Pagal oficialius npm'o doc'us:

      npm install npm@latest -g — paskutinis suveikęs

      Iš pirmo karto šitas kuo puikiausiai suveikė:

      https://docs.npmjs.com/getting-started/installing-node#updating-npm

      Node comes with npm installed so you should have a version of npm. However, npm gets updated more frequently than Node does, so you'll want to make sure it's the latest version.

      npm install npm -g

      O kiekvieno puslapio apačioje parašyta:

      The current stable version of npm is 3.10.6

      To upgrade, run: [sudo] npm install npm@latest -g

    2. O šitas npm'o neatnaujino:

      Iš http://nodejs.org/ atsisiųsti install'erį ir suinstaliuoti, suinstaliuos ir naujausią npm'ą (Ar tikrai?).

    3. npm install -g npm@latest worked for me on Windows8 stackoverflow.com/a/29023180/588759

    4. http://stackoverflow.com/a/27235748

      Upgrading for Windows Users

      Windows users should read Troubleshooting > Upgrading on Windows in the npm wiki.

      Upgrading on windows 10 using PowerShell (3rd party edit)

      The link above Troubleshooting#upgrading-on-windows points to a github page npm-windows-upgrade the lines below are quotes from the readme. I successfully upgraded from npm 2.7.4 to npm 3.9.3 using node v5.7.0 and powershell (presumably powershell version 5.0.10586.122)

      First, ensure that you can execute scripts on your system by running the following command from an elevated PowerShell. To run PowerShell as Administrator, click Start, search for PowerShell, right-click PowerShell and select Run as Administrator.

      Set-ExecutionPolicy Unrestricted -Scope CurrentUser -Force

      Then, to install and use this upgrader tool, run (also from an elevated PowerShell or cmd.exe):

      npm install --global --production npm-windows-upgrade
npm-windows-upgrade

    5. To update npm, the npm update npm -g command didn't work for me (on windows). What did work was reinstalling npm according to the documentation: "You can download a zip file fromhttps://npmjs.org/dist/, and unpack it in the same folder where node.exe lives." Make sure if you do this that you get rid of your previous installation first (though overwriting it will probably work ok...).
