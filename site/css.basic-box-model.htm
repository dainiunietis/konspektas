    Vidiniai box model'ės box'ai

    Yra keturi box'ai, galimo jų didėjimo tvarka:

    • content box
    • padding box — pagal išorinę kraštinę (padding edge)
    • border box — pagal išorinę kraštinę (border edge)
    • margin box — pagal išorinę kraštinę (margin edge)

    The specification describes the content box, padding box, border box, and margin box, each being defined by the edges of the content, padding, border, and margin respectively.
