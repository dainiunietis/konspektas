    JavaScript

    kodo lint'inimas

    Lint'ingo atlikimo būdai:

    • via your editor (live linting)
    • via your build process
    • via pre-commit hook in version control

     

    Ką lint'ingas taiso:

    • Kodo stilių-formatavimą: atitraukimai-indent'avimas, kintamųjų-funkcijų pavadinimų stilius (Code style)
    • JS kalbos sintaksę (Syntax)

     

    JS linter'iai:

    • ESLint — atrodo geriausias
    • JSHint
    • JSCS
