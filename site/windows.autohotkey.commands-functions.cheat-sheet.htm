    AHK script'o vykdymas

    Laiko kontrolė

    Pause [, On|Off|Toggle, OperateOnUnderlyingThread?]

    Sustabdo arba nuima sustabdymą ir vėl pradeda vykdyti script'ą, iš kurio iškviesta ši komanda. Sustabdymas nėra apribotas laiku, script'as vėl pradedamas vykdyti kai iškviečiama Pause Off arba Pause Toggle.

    Šis sustabdymas neįtakoja hotkeys'ų (t. y. F2::KasNors()), timer'ių

    Pauses the script's current thread

     

    Sleep, DelayInMilliseconds

    Sustabdo script'p vykdymą nustatytam laikui.
