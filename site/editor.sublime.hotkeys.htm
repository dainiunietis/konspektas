    Teksto-kodo paieška

    ctrl+f

    [Find > Find...]

    Atidaro ir aktyvuoja (reiškia kursorius yra ieškomo teksto laukelyje) "search panel" redaktoriaus lango apačioje. Jeigu prieš atidarant yra pažymėjimas — jis įkeliamas į ieškomo teksto laukelį, beje taip pat veikia jei panelė jau yra atidaryta.

    ctrl+f3 / ctrl+f4

    [Find > Find Previous / Next]

    Atgal / pirmyn ieško teksto kuris yra paieškos panelės laukelyje. Veikia ir kai paieškos panelė atidaryta, ir uždaryta. Pažymėtas tekstas veikimo neįtakoja.

    f3 / f4

    [C][find_under_prev] / [C][find_under] [Find > Quick Find]

    Ieško atgal / pirmyn pažymėto teksto, o jeigu nėra pažymėjimo — žodžio, kuriame yra kursorius (tame tarpe ir kai kursorius yra prie vieno iš žodžio šonų).

    Ieškotas žodis įvedamas į Find ir Replace panel'ės ieškomo teksto laukelį, tai padaro nepriklausomai atidaryta panelė ar ne.

    Iš esmės tai [Find > Find...] naudojimas, vienu paspaudimu įvedamas paieškos string'as ir įvykdoma paieška.

    ctrl+shift+f

    [Find > Replace...]

    Atidaro "replace panel" redaktoriaus lango apačioje. Ieškomo teksto laukelį veikia kaip [Find > Find...]

    ctrl+shift+f4

    [Find > Replace Next]

    ctrl+super+f

    [Find > Find in Files...]

    Paieškos veikimo nustatymai-opcijos:

    "Use Buffer"

    Nustato kur bus išvedami paieškos rezultatai:

    • Įjungta — naujame redaktoriaus lange-buferyje, tokiu būdu galima turėti ne vienos paieškos rezultatus atskiruose languose.
    • Išjungta — panelėje redaktoriaus apačioje, ją su Esc paslėpus, galima su [Find > Find Results > Show Results Panel] vėl atidaryti. Čia rodomi paskutinės paieškos rezultatai.

    package::FuzzyFileNav

    https://github.com/facelessuser/FuzzyFileNav

    Esminis skirtumas tarp FuzzyFileNav'o ir [Goto > Goto Anything…] panel'ių — FuzzyFileNav'o panel'ėje filtruojama vieno katalogo turinys, o [Goto > Goto Anything…] — viso projekto.

    FuzzyFileNav'o quick panel'ės filtravimo input'o veikimas

    Pačioje pradžioje įvedus:

    • \ arba / — FuzzyFileNav'o panel'eje atidaro-nukeliauja į šakninį katalogą disko, kuriame šiuo metu yra panel'eje browse'inama
    • ~/ — FuzzyFileNav'o panel'eje atidaro-nukeliauja į Windows'ų user'io katalogą (the home folder), pas mane šiuo metu — C:\Users\zm

    ctrl + f will open a folder search panel with the current folder’s path, or a the path of a subfolder if one is typed into the FuzzyFileNav panel, pre-loaded into the where box. Any content in clipboard will be pre-loaded into the Find box.

     

    Operacijos su failas

    ctrl+d

    ctrl + d deletes the folder/file object currently typed in the FuzzyFileNav panel.

     

    ctrl+c -- ctrl+x -- ctrl+v

    Failų kilojimas tarp katalogų su Copy, Cut, Paste

    Fuzzy File Copy

    ctrl + c Copies the folder/file object currently typed file in the FuzzyFileNav panel. The copy will remain in the clipboard until a paste is performed or a new copy is initiated.

    Fuzzy File Cut

    ctrl + x cuts (moves) the folder/file object currently typed into the FuzzyFileNav panel. A Fuzzy File Paste must be performed to complete the cut (move).

    Fuzzy File Paste

    ctrl + v Completes the copying and pasting of the folder/file object that is in the clipboard. The file/folder will be pasted into the currently opened folder in the FuzzyFileNav panel. To rename the folder/file object on paste, type the full name in the panel that should be used before pressing ctrl + v.

     

    Naujų failų-katalogų sukūrimas

    Fuzzy Make File

    ctrl + n Creates a new file in the currently opened folder in the FuzzyFileNav panel. The name that is typed into the panel is the name that will be used.

    Fuzzy Make Folder

    ctrl + shift + n Creates a new folder in the currently opened folder in the Fuzzy File Nav Panel. The name that is typed into the panel is the name that will be used.

    Simbolių kilojimas-perstumdymas

    ctrl+t

    [Edit > Text > Transpose]

    Kai nėra pažymėjimo, yra vienas, ar multiple kursoriai

    Sukeičia vietomis šalia kursoriaus esančius simbolius.

    Kai yra vienas pažymėjimas

    Nieko nedaro.

    Kai yra du ar daugiau pažymėjimų

    Sukeisdamas vietomis suka ratu pažymėjimus, t. y. pirmasis pažymėjimas atsiranda antrojo vietoje, antrasis trečiojo, ir taip toliau, o paskutinis stoja į pirmojo vietą.

    Pažymėjimo pozicija nustatoma pagal poziciją kode, pačių pažymėjimų darymo seka įtakos neturi.

    Kai yra ir pažymėjimų, ir kursorių, nesvarbu ko daugiau ar mažiau, svarbu, kad vienu metu yra bent vienas pažymėjimas ir bent vienas kursorius

    Kursoriai expand'inasi ir pažymi žodį kuriame yra (jeigu yra tarp dviejų ar daugiau tarpelių, pažymi juos), ir veikia taip pat kai yra du ar daugiau pažymėjimų.
