The solution is to combine opacity and visibility. Unlike the display property, you can apply a transition to the visibility, but you shouldn't transition from 'visibile' to 'hidden' because, once again, we don't have values in between and some browsers will get confused. However, you can set a transition-delay: after the opacity transition is over, set visibility hidden. Done!
