    CSS Writing Modes Level 3

    writing-mode:

    Nustato du parametrus:

    inline direction (axes)

    Nustato kokia ašimi — horizontaliai ar vertikaliai — atvaizduojant dėliojami:

    • teksto raidės ir kiti simboliai,
    • inline-level elementai-container'iai.
    block direction (axes)

    Nustato kokia ašimi ir kryptimi atvaizduojant dėliojami / block flow direction:

    • teksto simbolių eilutės,
    • block-level elementai-container'iai.

    Nustato teksto simbolių dėliojimą:

    • kaip tekstas rašomas: horizontaliai ar vertikaliai, t. y. raidžių ir kitų simbolių dėliojimo ašis,
    • ir kokia kryptimi dėliojamos teksto eilutės.

    The writing-mode CSS property sets whether lines of text are laid out horizontally or vertically, as well as the direction in which blocks progress. 

    Taip pat nustato block-level ir inline-level elementų-container'ių dėliojimo ašį ir kryptį:

    • inline-level — dėliojimo ašis kaip teksto simbolių,
    • block-level — dėliojimo ašis ir kryptis kaip teksto eilučių.

    This property specifies the block flow direction, which is the direction in which block-level containers are stacked, and the direction in which inline-level content flows within a block container. 

     

    Geroji praktika

    When set for an entire document, it should be set on the root element (html element for HTML documents).

     

    writing-mode: horizontal-tb — default'inis nustatymas, ką reiškia reikšmė:

    • horizontal — tekstas rašomas horizontaliai, krypties: left-to-right ar right-to-left nenustato
    • tb — Top-to-Bottom, teksto eilutės dėliojamos iš viršaus į apačią

     

    writing-mode: horizontal-tb — skirta kalboms:

    This covers all Western and some Middle Eastern languages, which may differ in the direction of their horizontal writing.

    Konkrečiaus:

    Western languages

    For ltr scripts, content flows horizontally from left to right.

     

    Hebrew and Arabic

    For rtl scripts, content flows horizontally from right to left. The next horizontal line is positioned below the previous line.

     

     

    writing-mode: horizontal-tb — default'ė reikšmė

    The default value, horizontal-tb, means "a horizontal inline direction, and a top-to-bottom block direction."
