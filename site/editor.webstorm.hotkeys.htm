    Visos [Window > Active Tool Window > ...] komandos veikia Tool Window'są kuris:

    • šiuo metu aktyvus

    Esc

    Esant aktyviam Tool Window'sui gražina fokusą redaktoriui ir jo neuždaro.

    shift+Esc

    [Window > Active Tool Window > Hide Active Tool Window]

    Kai aktyvus:

    kuris nors Tool Window'sas
    jį uždaro ir gražina fokusą redaktoriui
    redaktorius
    uždaro paskutinį aktyvų Tool Window'są, fokusas išlieka redaktoriuje

    [Window > Active Tool Window > Hide Side Tool Windows ]

    Kai aktyvus:

    kuris nors Tool Window'sas
    uždaro visus tos pusės Tool Windows'us ir gražina fokusą redaktoriui
    redaktorius

    uždaro paskutinio aktyvaus Tool Window'so pusės visus Tool Windows'us, fokusas išlieka redaktoriuje

    dar kartą įvykdžius — uždaro priešpaskutinio Tool Window'so pusės visus Tool Windows'us
