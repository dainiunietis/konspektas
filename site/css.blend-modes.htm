    Naudingiausi:

    You'll find that the most useful blend modes for pleasing results are darken, multiply, overlay, screen and soft-light.

    Blend Mode'ai tagams

    mix-blend-mode:

    mix-blend-mode | CSS-Tricks

    Visas tago turinio spalvas blend'ina su viskuo kas yra po juo:

    The mix-blend-mode CSS property sets how an element's content should blend with the content of the element's parent and the element's background.

    Sukuria new stacking context:

    It's worth noting that setting a blend mode other than normal will generate a new stacking context which must then be blended with the stacking context that contains the element.

    Another spot where isolation seems to be invalidated is when it is used in conjunction with translate on the same element. You might bump into this if you attempt to center an element both vertically and horizontally using absolute positioning and translate together:

    .module {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  isolation: isolate;
}

    The use of translate appears to isolate the element on its own without the use isolation.
